# Rabota_Slavy — Система управления оконным производством

**JavaFX-приложение для управления продажами и производством окон.**

## Назначение
- Каталог комплектующих (Rehau, Roto NX и др.)
- Оформление заказов с оплатой
- Авторизация / регистрация / режим гостя
- Чат с поддержкой (сохранение в PostgreSQL)
- Личный кабинет

## Стек технологий
- Java 17 + JavaFX 17 + FXML + CSS
- PostgreSQL + JDBC + DAO
- jBCrypt (хеширование паролей)
- Maven + Git

## Установка
1. `git clone https://github.com/pinkiePie657/Rabota_Slavy.git`
2. Создать БД `okno` в PostgreSQL и выполнить SQL-скрипт
3. Запустить `Launcher.java`

## Использование
- Можно работать как гость
- Выбрать товар → оплатить (валидация карты)
- Чат с поддержкой сохраняется в БД
